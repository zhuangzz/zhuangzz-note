模块入门

模块（ module ），基本意思就是把一个大程序，切分成多个文件。这样，说白了，一个文件就是一个模块。但是，在 ES6 编程领域，模块有自己的一些固定的使用语法。

ES6 模块

react 写代码的时候要用 ES6 模块。

es6 module 基本语法如下：

export // (命名导出,默认导出)
import // 导入
