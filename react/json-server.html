JSON-Server安装

作为前端和客户端开发人员，在后端还没有给出对应的api接口时，我们无法做测试。 这时，我们可以使用json-server快速搭建一个测试的api接口，能快速搭建好。

创建db.json

在里面加入json格式的数据

{
  "post":[
{
  "id" :"13479587123",
  "title" :"标题一",
  "content":"nksdjnkj"
},
{
  "id":"264564123",
  "title":"标题二",
  "content":"nksdjnkj"
},
{
  "id":"37165756723",
  "title":"标题三",
  "content":"nksdjnkj"
},
{
  "id":"476571233224",
  "title":"标题四",
  "content":"nksdjnkj"
},
{
  "id":"5123dsd",
  "title":"标题五",
  "content":"nksdjnkj"
}]
}

全局安装JSON-Server

npm install -g json-server

运行JSON-Server

json-server --watch db.json -p 3008
