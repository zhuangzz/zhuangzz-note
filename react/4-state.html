第四节:命令行

ctrl alt t 深度系统打开终端

参考http://newming.coding.me/myidoc/html/%E5%9F%BA%E7%A1%80%E7%9F%A5%E8%AF%86/vi%E7%BC%96%E8%BE%91%E5%99%A8%E6%93%8D%E4%BD%9C.html

Command + K 清屏

Command + T 新建标签

Command +W 关闭当前标签页

Command + S 保存终端输出

Command + D 垂直分隔当前标签页

Command + Shift + D 水平分隔当前标签页

Command + shift + {或}向左/向右切换标签

1、将光标移动到行首：ctrl + a

2、将光标移动到行尾：ctrl + e

3、清除屏幕： ctrl + l

4、搜索以前使用命令：ctrl + r

5、清除当前行： ctrl + u

6、清除至当前行尾： ctrl + k

7、单词为单位移动：option + 方向键

killall Dock

pwd 当前位置

ls 查看当前目录下所有内容

cd 跳转目录

cd 1706/txt

cd ../返回一级

cd ../../返回两级

cd .当前级

cd c:跳到c盘

cd ~根目录

ls -a 查看当前目录下所有东西，包括隐藏文件

ls --help帮助命令

mkdir 创建文件夹

mkdir aa bb cc建三个文件夹

mkdir aa/bb在aa中创建bb

cd aa/cc && mkdir a b c 命令行连用在cc中创建a b c

touch index.html 创建文件

touch cc/inde.html

ctrl c 结束当前命令

rm a.doc删除文件

rm b.ppt b.avi a.exe

rm -r cc 删除文件夹和文件

rm -r ./* 删除当前目录下所有东西

rm -rf 强制删除

cp 复制

cp -r aaa bbb 复制aaa叫bbb（复制文件不用加-r，复制文件夹就要加-r）

cp bbb/test.txt test1.txt 将test.txt复制到当前文件夹名字叫test1.txt

cp bbb/e.txt aaa/b.txt 将bbb/e.txt复制成aaa/b.txt

cat bb/inde.html 查看当前文件下内容

mv t.txt bb/tf.txt 移动、剪切(如果在同一个文件夹下，相当于重命名)

mv t.txt ../bb/

mv bb/f.txt aa/ 将f.txt移动到aa中

ctrl n ctrl s
