第四节:命令行

ctrl alt t 深度系统打开终端

pwd 当前位置

ls 查看当前目录下所有内容

cd 跳转目录

cd 1706/txt

cd ../返回一级

cd ../../返回两级

cd .当前级

cd c:跳到c盘

cd ~根目录

ls -a 查看当前目录下所有东西，包括隐藏文件

ls --help帮助命令

mkdir 创建文件夹

mkdir aa bb cc建三个文件夹

mkdir aa/bb在aa中创建bb

cd aa/cc && mkdir a b c 命令行连用在cc中创建a b c

touch index.html 创建文件

touch cc/inde.html

ctrl c 结束当前命令

rm a.doc删除文件

rm b.ppt b.avi a.exe

rm -r cc 删除文件夹和文件

rm -r ./* 删除当前目录下所有东西

rm -rf 强制删除

cp 复制

cp -r aaa bbb 复制aaa叫bbb（复制文件不用加-r，复制文件夹就要加-r）

cp bbb/test.txt test1.txt 将test.txt复制到当前文件夹名字叫test1.txt

cp bbb/e.txt aaa/b.txt 将bbb/e.txt复制成aaa/b.txt
