第四节：Redux Store

Redux 是数据流管理工具。使用 Redux 的最重要的一句话：

一切数据都要保存的 Store 之中，组件自己不保留自己的 state 数据

为何要统一存放到 Store 中？

  • 第一，各个组件都统一读写同一个数据，组件间通信变得不成问题了

  • 第二，就是数据统一存放，代码比较不容易写乱

    注：将代码回滚到第二节完成后。

创建 Store

store 中的数据，必然要涉及读和写的操作，所以我们要安装 一个包来辅助后续操作：

npm i --save redux
