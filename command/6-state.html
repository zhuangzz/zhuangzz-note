第六节:nodejs安装

nodejs 使javascript可以脱离浏览器运行，可以在node上或者本地运行

nvm是 node版本管理工具

安装brew

  • curl -LsSf http://github.com/mxcl/homebrew/tarball/master | sudo tar xvz -C/usr/local --strip 1

  • 安装完成后执行brew

  • 赋给/usr/local目录权限，执行sudo chown -R $(whoami) /usr/local（即sudo chown -R fxp /usr/local其中fxp为系统当前用户名）

  • brew update

  • 按照提示删除/usr/local/share/doc/homebrew，执行 rm -r -f /usr/local/share/doc/homebrew

  • 再次执行brew update

brew使用

  • brew install curl

  • brew install git

nodejs安装

curl安装

  • 深度sudo apt-get install curl

  • Macbrew install curl

安装nvm

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.29.0/install.sh | bash

(如果不能用，就nvm use stable)

安装node

查看有哪些版本可以安装
nvm ls-remote
安装版本 v5.10.1
nvm install v5.10.1

node tt.js可以得到tt.js的运行结果
