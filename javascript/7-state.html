第七节:函数 正则表达式

函数相关

var arr=[4]
console.log(arr.push(1))//输出长度2

    console.log(a)//function a(){return 4}
    var a=1
    function a(){
        return 4
    }
    console.log(a)//1

    var num=10
    console.log(10===num)//true值的类型
    var arr1=[1,2]
    var arr2=[1,2]
    console.log(arr1==arr2)//false引用类型（对象）

正则表达式

var re=/a/
var str="axb"
re.text(str)// 返回true或false
re.exec(str) //返回类数组（正确匹配类信息）或null
console.log(re.exec(str)[0])//a

var re=/[abc]/  
var str="asdhjc"
re.text(str)//true
re.exec(str)//

var re=/[0-9]/   //[0-9] 一位数字

[]一位

  • [0-9] 一位数字

  • [a-z]一位小写字母

  • [A-Z]一位大写字母

  • [A-z]所有字母和下划线

  • [A-z0-9张]一位字母数字下划线（不包括中滑线）

+ ? * 只能控制前面一位

  • []+ 1位或多位 []{1,}无穷 a+

  • []? 0位或一位 []{0,1}

  • []* 0或多位 []{0,}

  • []*? 0位 []{0}

  • []+? 一位 []{5} []{2,5}

  • (?=str)紧跟着str

  • (?!str)不紧跟着str

  • \b 数字 匹配单词边界 写在哪边就匹配哪边

  • \B非数字

  • \w数字字母下划线

  • \W非数字字母下划线

  • \s一个空白符：空格 回车 tab

  • \S非空白符

  • ^ 写在中括号里非a |或 写在中括号外开始

  • $ 结尾

  • \转义符把在js中写不上的转义

i不区分大小写 g全局匹配不在第一次匹配后停止

/gi == /ig
