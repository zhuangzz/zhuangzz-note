驼峰命名法

变量声明提升 只要声明了的变量都会把声明（例：var num） 提升到当前作用域的最顶端

var num;
console.log(num);
num=10//最后输出undefind

当变量没有赋值时，undefind就是它的值

无论声明了几个变量，变量声明是同步执行的

数据类型的转换

字符串和数字之间的转换

  • string->number Number(string) parseInt(string) parseFloat(string)

Number ： 先看string整体是不是数字，是的话转换，不是的话转换成Nan(也是数字类型，通过不正当的转换过来的)

parseInt：转换成整数，先看第一个字符是不是数字，是就转换，直到不是数停止，假如第一个不是数字转换成Nan

parsefloat：转换成浮点型

var bool=true;
console.log(Number(bool))//布尔值转化数字

    var num1="";
    var sign="";
    var num2="";
    var result="";
    $(".number button").click(function(){
        var num=$(this).text()
        if(sign==""){
            num1+=num
            $("input").val(num1)
        }else{
            num2+=num
            $("input").val(num1+sign+num2)
        }

    })
    $(".sign button").click(function(){
        var sign =$(this).val()
        $("input").val(num1+sign)
    })
    $(".result").click(function(){
        if(sign){
            result=eval(num1+sign+num2);
        }
        $("input").val(result)
    })
</script>
