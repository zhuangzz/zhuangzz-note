UTF-16

  • str.charAt(0)一个字符串中第1个字符
  • str.charCodeAt(1)一个字符串中第2个字符的UTF-16码
  • str.concat(string2, string3[, ..., stringN])连接一个或多个字符串
  • str.endsWith("a",b)检测str字符串是否以a结尾，根据判断结果返回 true 或 false。b参数规定检测的末端位置（b是第b个字符） es6新特性
  • str.includes(“a”,1)检测a是否在字符串中，从索引值为1的开始
  • str.indexOf(“a”,1)检测a字符串在str中是否存在，存在返回第一次出现的位置，不存在返回-1。从索引值1开始找
  • str.link(url) 创建一个href的a标签 str当作标签内部的文字
  • str.repeat(6)返回新的重复之后的字符串，6代表重复6次
  • str.slice(1, 4)从字符串中截取一段 提取新字符串从第二个字符到第四个 (字符索引值为 1, 2, 和 3)。 str.slice(2, -1) 提取第三个字符到倒数第二个字符。

  • str.split("") 方法将一个String对象分割成数组，通过"" " " "-"等将字符串分成子串。获取字符串中单个字符出现次数最多

  • str.toLocaleLowerCase()返回字符串值转换为小写的值

  • str.toLowerCase()返回字符串值转换为小写的值

  • str.toLocaleUpperCase()返回字符串值转换为大写的值
  • str.toUpperCase()返回字符串值转换为大写的值

  • str.trim()将字符串的两端空白字符删除

  • str.trimLeft()删除左边空白

  • str.trimRight()删除右边空白
