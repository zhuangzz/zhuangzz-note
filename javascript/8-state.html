match replace search

str.match(a|regexp)检测str中是否有字符a或是否有符合正则表达式的字符串，返回数组或null

var str="hello world"
var re=/[l]/g/
console.log(str.match(re))//"l","l","l"

str.replace(a|regexp,b)将str字符串中的a或与正则表达式匹配的内容 替换成b

var str = 'before Xmas';
var newstr = str.replace(/xmas/i, 'Chris');
console.log(newstr);  //before Chris

search str.search(a|regexp,b)检测str中是否有子字符串a或是否有符合正则表达式的字符串 返回第一次匹配的位置或-1

var str="hello"
console.log(str.search(/h/))
