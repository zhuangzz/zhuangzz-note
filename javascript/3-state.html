第三节:对象 数组 数学

对象

内置对象（数字 字符串 布尔 数组 正则表达式 数学 日期 函数）

自定义对象

js对象中包含两个对象： 属性 方法

数组对象（array）数据的集合

var arr =[];//创建一个数组

var arr=[1,2,3,4]//里面的数据类型可以随意

arr[3]//获取数组里的某一项

arr[2]=4//修改数组第三项内容，重新赋值，也可以用此方法添加新项

数组属性 length数组的长度 格式：对象名.属性名 对象.属性名

arr.length

[1].length

数组的方法 对象控制的功能或方法 格式：对象.方法名(参数)

.push()向数组内部最后添加元素,并返回新数组的长度

只有方法有返回，方法执行之后，整个方法变成返回值，可以根据返回值去作出判断

  • arr.push(4,5,6,[7])

  • arr.push(arr1)

  • .pop()删除后返回数组中最后一个元素

  • .shift()删除后返回数组中的第一个元素

  • .unshift()向数组内部开头添加一个或多个元素

  • arr1.concat(arr2,arr3)合并两个或多个数组，并返回新数组

  • .reverse()颠倒数组，倒序 回文（正反读都一样）

  • .slice(放数组下标或索引值)截取数组，返回截取的数组，从索引a开始（包括a）,到b结束（不包括b）

  • .join(a)数组的所有元素放入一个字符串，通过指定的分隔符a进行分割，分隔符默认是逗号，要用空隔开可以用""。console.log(arr1.join("ww")//加双引号，不然会以为是变量)
  • .splice(1，2，3)向数组中添加或删除元素，（1是删除和添加的位置，2是删除的个数0~n，3添加的元素,3可以不写），返回被删除的数组var num5=arr.splice(1,2,3)
  • .sort()排序，返回新数组

arr.sort(function(a,b){return a-b})从小到大排序
