for循环

for(var i=0;i<10;i++){ console.log(i) }

  • 声明变量i，并初始值为0

  • 判断i<10是否成立

  • 判断成立时执行大括号内的内容 执行i++ 并返回第二步

  • 不成立跳出循环

        var x=0;
        for(i=0;i<10;i++){
            console.log(i)
            x=x+i

        }
        console.log(i)
        console.log(x)
