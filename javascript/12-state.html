this undefined

this指向的是一个对象。哪个对象调用包含this的方法，方法内部的this 就指向那个对象。在行内写事件的时候，事件函数里面的this指向的不是这个元素本身而是window

var a=20
function fun(){
    var a=10
    console.log(this.a)//20 window.a
}
fun()//window.fun()
console.log(window.a)//20

出现undefined的情况：变量声明没有赋值。获取一个对象的属性值的时候，这个对象没有这个属性

<button onclick=fun()>anniu</button>//this是window

    console.log(this)
}
document.querySelector("button").onclick=fun;//this是<button></button>
